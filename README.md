## Integration
### CocoaPods (iOS 8+)
You can use [CocoaPods](https://cocoapods.org/) to install _SHGradientColorView_ by adding it into your _Podfile_.

    use_framework!

    pod "SHGradientColorView"

- Remember to add _use_framework!_ in your _Podfile_.
